# ✨ Markov Chain Text Generator (Advanced)

## 📌 Project Overview
This project demonstrates text generation using a **Markov Chain with Bigram (N-gram) model** in Python. It generates new text by learning patterns from sample input data and predicting the next word based on previous words.

---

## 🧠 Concept
A Markov Chain is a statistical model that predicts the next word based only on the current state.

In this project:
- We use a **Bigram model (2-word combination)**
- Each pair of words is used to predict the next word
- This produces more natural and meaningful text than a basic Markov Chain
